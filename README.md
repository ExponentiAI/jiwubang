# jiwubang
## 急物帮：新型肺炎生活（应急）物资小帮手。

![](https://github.com/ExponentiAI/jiwubang/blob/master/JWB_UI/jiwubang000.png)

针对 2020 年初在武汉爆发的新型冠状病毒疫情，本项目旨在收集居民和群众反馈的周边超市、药店卖口罩和消毒液的情况（位置、门店、基本类别、价格），便于这阶段居民出门购物和找到合适的地方购买口罩和消毒液。
